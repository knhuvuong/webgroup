PHẦN 1: GIỚI THIỆU

Firebase là gì?

► Firebase là một nền tảng do google cung cấp hoạt động trên cơ sở đám mây- Cloud để phát triển ứng dụng di động và trang web, bao gồm các API đơn giản và mạnh mẽ mà không cần backend hay server.

Tên miền là gì?

► Tên miền là tên của một website hoạt động trên internet, đóng vai trò là một địa chỉ tĩnh, cố định. Nó giống như là địa chỉ nhà hay mã zip code để giúp các thiết bị định tuyến vệ tinh dẫn đường cho hàng hóa lưu thông, một trình duyện cũng cần một tên miền để dẫn đường tới nơi chứa website của bạn (web server)..

PHẦN 2: TẠO WEB APP BẰNG FIREBASE

Bước 1: Cài đặt Note Js

Cài đặt Node Js tại đây

Bước 2: Cài đặt FireBase CLI

1. Mở Command line bằng tổ hợp "Window + R"

2. Nhập lệnh npm install -g firebase-tools

3. Sau khi cài xong để kiểm tra chạy lệnh "firebase --version"
Nếu như hiện phiên bản như bên dưới hoặc cao hơn là đã thành công.

Bước 3: Tiến hành tạo một trang web

1. Vào trang web firebase.google.com.

2. Đăng nhập vào FireBase bằng tài khoảng google và chọn chọn get start.

3. Tạo một Project mới.

4. Đặt tên cho Project và chọn Continue

5. Chọn Continue.

6. Chọn Default Account for Firebase và chọn Continue.

7. Click vào đây để tạo Website cho Project của bạn.

8. Đặt tên Website cho Project và chọn "Register the application"

9. Vậy là bạn đã đăng kí thành công một web app trên Firebase để chứa Project

10. Tạo tiếp một thư mục trên máy tính của bạn để chứa dự án

11. Mở "cmd" trong thư mục vừa tạo

12. Đăng nhập vào Firebase bằng lệnh firebase login

13. Tiếp tục gõ lệnh firebase init hosting để kết nối CLI tới tài khoản Firebase online.

14. Chọn "Use an existing project" và chọn tên dự án bạn đã tạo ở trên.

15. Trả lời các câu hỏi và chờ hiện ra "Firebase initialization complete!".

16. Tiếp theo cần gõ lệnh firebase deploy để đưa đưa code lên hosting của firebase là xong.

17. Truy cập vào đường link trong mục "Project console" hoặc "Hosting URL" để vào website của bạn.

18. Vậy là bạn đã tạo xong trang web để chứ Project !!

PHẦN 3: KẾT NỐI TÊN MIỀN

Bước 1: Chọn 1 tên miền

1. Để kết nối tới trang web chúng ta cần 1 đường link, vào Freenom để build 1 tên miền miễn phí

2. Nhập tên miền và phần đuôi muốn mua và nhấn Enter để kiểm tra tên miền

3. Click vào "1 domain in cart", chọn tháng muốn sử dụng, nhấn "check out"

4. Cần đăng nhập để "Check out", nếu chưa có thì đăng kí

5. Click vào logo để trở về trang chủ và chọn: "Service -> My Domain"

6. Click chọn "Manage Domain" trên tên miền đã chọn
7. Giao diện như hình, nhấn sang phần "Manage Freenom DNS"

Bước 2: Kết nối tên miền

1.Vào firebase bằng chrome, chọn project đã dùng để tạo web, nhấn thanh menu chọn mở phần "hosting", chọn "get Started"

2. Chọn "Add custom domain"

3. Sao chép mục "Target" ở phần tên miền, chuyển giá trị mục type thành "txt"
.

5. Nhấn "Saves changes", tạo thêm 2 mục records

6. Ở trang firebase, nhấn "verify" sẽ bị lỗi, đợi 5-10p để hệ thống lưu rồi mới verify được

7. Sau khi đã verify, sao chép 2 value còn lại vào mục target

8. Nhấn lưu lại

9. Hiện tại tên miền đang ở trạng thái "need setup", cần chờ từ 30p-24h để tên miền được kết nối là hoàn thành.
10. Vậy là bạn đã hoàn thành kết nối tên miền cho website !
